# IsometricDx
Isometric Game in C++, using SDL

The focus was to make the engine expandable to introduce new features later on.

Features:
  - Skeleton enemy
  - Projectile Shooting
  - Obstacles (ingame)
  - Menus and Music
  - Obstacles (in code)
  - and of course a likeable player character
